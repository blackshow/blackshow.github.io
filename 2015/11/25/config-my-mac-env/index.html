无废话Mac环境配置

Everybody should use Mac.

拿到一台新Mac首先要做什么

拿到一台新的Mac，应该先做下面这些事:

  • 左上角  > 关于本机 > 软件更新
  • 安装iTerm 2(Mac下最好的终端替代软件)
  • 配置右键直接在finder中打开当前路径的终端

  • 隐藏Dock
    可以获得更大的工作空间，这里要把Dock唤出的延迟去掉

  • 隐藏菜单栏
    隐藏了之后由于看不到时间，工作格外专注

  • 安装Homebrew(Mac OS X下最好的包管理工具)
    Homebrew的简介和基本操作
  • 安装Homebrew-cast(一个简洁、优雅、迅速的Homebrew扩展)
    Homebrew Cask 的简介和基本操作
  • 安装CleanMyMac
    如果你和我一样是强迫症的话，请使用CleanMyMac。
  • 安装Parallel Desktop(如果你和我一样有些不得不用windows的原因)
    Mac OS X下融合模式最好的虚拟机软件

做完这些，基本上环境搭建已经成功了一大半了。

开始安装软件

安装软件的原则是：

能通过Homebrew安装的就用Homebrew和Homebrew Cast安装。

开发工具

by Homebrew

  • git
  • git-flow
  • maven
  • mysql
  • redis
  • wget
  • zsh
  • tomcat
  • nvm
  • ssh-copy-id(自动把公钥部署到服务器)
  • …

    by Homebrew-cask

  • java6
  • java7
  • java(java8)

普通安装

  • IDEA Intellij
  • WebStorm
  • Android Studio
  • Genymotion
  • Xcode
  • Dash(很牛逼的Api查询工具)
  • Sublime text 3
  • Remote Desktop Connection(总是避免不了要远程桌面连windows服务器的情况)

生产力工具

  • Alfred 2(搜索工具)
  • 印象笔记(更新了OS El Capitan之后，系统默认的note.app就很好用了)
  • 百度云同步盘(不能选择目录进行同步，否则闪退)
  • SizeUp(窗口管理神器，El Capitan自带的分屏操作跟这个根本没法比。。)
  • Keka(免费的压缩软件，除了rar全支持，你懂的)
  • ProxyMenu(快速切换代理配置，Safari没有类似Chrome中SwitchSharp功能的插件，用这个是最方便的了)
  • 搜狗输入法
  • Caffeine(给Macbook嗑点药，防止电脑休眠)

办公工具

  • Microsoft Office

其他

  • KeyUp(屏幕虚拟键盘，还能模拟机械键盘的声音，炒鸡炫酷)
  • PixelPeeper(监视屏幕上一个像素点颜色变化的软件，发现颜色变化以后支持ActionScript，可以报警、关机、休眠、发邮件等等)
  • 迅雷 for Mac(总得有个下载软件。。。)
  • VLC（解码库很全的视频播放器）
  • CheatSheet(按住command显示快捷键,Mac 新人必备)

online tools

  • http://alternativeto.net
    如果你从Windows转到Mac OS X后，在Mac上找不到曾经在和Windows上用过的软件功能相似的替代品，试试这个网站吧。
  • SVG画图
    在线画图，可以直接导入图片转换成svg。
  • 像素风格ico生成器
    本站的logo和favico就是用这个网站画的，但是只支持生成16x16的，要想获取其他尺寸自己PS一下吧
  • 图好快
    图片压缩网站
  • 在线UML
    在线UML绘图工具,速度还可以

如果你有好的软件，无论是做什么用的，欢迎在评论中留言告诉我，谢谢。
